# Two Sum

## Problem Statement

Given an array of integers `nums` and an integer `target`, return `true` if there are two numbers that add up to `target` and `false` otherwise.
You may not use the same element twice.

## Example

- nums = [2, 7, 11, 15], target = 18
  - Output: `true`
- nums = [3, 2, 4], target = 6
  - Output: `true`
- nums = [3, 3], target = 6
  - Output: `true`
- nums = [3, 4], target = 6
  - Output: `false`
